# Changelog

Reworked the target system
- The player must now pick up a "package", which must then be brought to a "destination" to complete a "delivery"
- Each completed delivery increments the score by one

Added a timer that triggers a "game over" state upon expiration
- upon entering game over, active packages/destinations are despawned, and new ones can not spawn

Added UI elements for deliveries completed, time remaining, and game over

Added several new chunk designs for road generation

Added a background track: Dmitri Shostakovich - Waltz No. 2 (I just had it in my downloads folder already)

Added sound effects for picking up packages and completing deliveries
